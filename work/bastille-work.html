The Challenge

Due to the nature of IOT security (or lack thereof), one of the biggest challenges did not revolve around the product directly. A bigger challenge was creating awareness and educating people around the fact that most IoT devices lacked basic security, which in turn acted as a gateway for potential hackers. This only became a bigger issue as more and more IoT devices got pushed to market. Through research we were able to construct put together various use cases which in turn allowed for us to build a product around.

Enterprise Dashboard

Along with a physical sensor that would be deployed within a corporate environment, the sensors would transmit information into the cloud which would then display via a UI for analysis. This afforded security professionals with the ability to get full insight into what was happening on their network at all times.
